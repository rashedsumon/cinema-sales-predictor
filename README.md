# Cinema Ticket Sales Predictor

An AI-powered web application built with Streamlit that predicts cinema ticket sales based on historical data. It automatically downloads the required dataset from Kaggle Hub on its first run.

## How to Deploy on Streamlit Cloud

1. Push this entire repository to your GitHub account.
2. Log in to [Streamlit Share](https://share.streamlit.io/).
3. Click **New app**, select your repository, branch, and set the Main file path to `streamlit_app.py`.
4. Click **Deploy!**
